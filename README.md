# Snippetly — Pastebin Clone using Flask & SQLite3

A Simple and Fast Way to Share Code Snippets and Text Online

## Overview

Snippetly is a lightweight and user-friendly platform that allows users to share code snippets or plain text instantly and securely.
Inspired by the concept of Pastebin, the project is designed with simplicity and speed in mind — making it perfect for developers, students, and anyone who needs to share quick notes or code blocks online.

With features like snippet expiration, clean UI, and easy sharing, Snippetly provides a smooth user experience and serves as an excellent real-world example of a Flask-based web application.

## Features
### 1. Create and Share Snippets

* Add any text or code instantly
* Save and generate a unique link for each snippet
* Easily share with others

### 2. Syntax Highlighting (Optional Enhancement)

* Supports multi-language highlighting

* Improves readability for developers

### 3. Paste Expiration

* Set custom expiration time

* Automatically deletes snippets after expiry

### 4. Clean & Minimal UI

* Modern responsive design

* Simple and intuitive interface

## Tools & Technologies Used
| Tool / Technology | Purpose                                   |
| ----------------- | ----------------------------------------- |
| **HTML**          | Page structure and layout                 |
| **CSS**           | UI styling and responsiveness             |
| **Flask**         | Backend framework, routing, logic         |
| **SQLite3**       | Lightweight database for storing snippets |

## Project Structure

* Home Page: Intro and overview of Snippetly

* Create Snippet Page: Add and save code/text snippets

* View Snippet Page: Displays shared snippet

* About Page: Information about the project
  
## Conclusion

The Snippetly project demonstrates how simple web technologies can be combined to build a functional and elegant code-sharing platform.
With clean UI, easy sharing, and a reliable backend, the aim is to offer a smooth and efficient experience for anyone who wants to store or share text snippets online.

This project showcases a complete pipeline — from front-end interface to back-end logic and database storage — making it a perfect beginner-to-intermediate Flask project.
